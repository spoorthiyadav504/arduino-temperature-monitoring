Arduino UNO - Temperature Monitoring System

Project Name

Temperature Monitoring System using Arduino UNO

Domain

Embedded Systems and IoT

Description

This project is based on Arduino UNO and LM35 temperature sensor. The system reads temperature values from the sensor and displays them through the Serial Monitor.

Components Used

- Arduino UNO
- LM35 Temperature Sensor
- USB Cable
- Arduino IDE

Working Principle

The LM35 sensor provides an analog voltage proportional to temperature. Arduino reads the analog value from pin A0, converts it into temperature in Celsius, and displays the result on the Serial Monitor.

Output

Real-time temperature values are displayed in Celsius.
